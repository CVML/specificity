# Image Specificity

These scripts are intended to reproduce the results in the paper "Image Specificity".

- [arXiV preprint](http://arxiv.org/abs/1502.04569)
- [supplementary material](https://computing.ece.vt.edu/~mainak/webpage/)
- project webpage
