Data used in "Understanding the intrinsic memorability of image." Isola, Parikh, Torralba, and Oliva. NIPS 2011.

anno_feats is a MATLAB array with values for each of the 923 annotation types and 2222 images used in our experiments.

anno_names is a list of names that describe the 923 annotation types, in the same order as anno_feats.
